# Сайт КиберПапа — пакет для выкладки

Дата сборки: 20 апреля 2026 г.

---

## 📦 Что в архиве

```
kiberpapa-site/
├── index.html                 ← главная страница сайта
├── cyberpapa-hero.jpg         ← фото КиберПапы для hero
├── README.md                  ← этот файл
├── crisis-protocol.md         ← протокол обработки кризисов в боте (в репо, не на сайте)
└── legal/
    ├── terms.html             ← Пользовательское соглашение
    ├── privacy.html           ← Политика конфиденциальности (по 152-ФЗ)
    ├── offer.html             ← Публичная оферта
    └── refund.html            ← Политика возврата
```

Все 5 HTML-страниц связаны между собой:
- На главной в футере — ссылки на все 4 документа
- Внутри каждого документа — кнопка «На главную» + навигация между документами

---

## ⚡ Быстрый старт — посмотреть сайт локально

1. Распакуйте архив в любую папку.
2. Двойной клик по **`index.html`** — откроется в браузере.
3. Проверьте ссылки в футере — должны открываться страницы документов.

**Важно:** сохраняйте **структуру папок** как в архиве. Файлы `*.html` в `legal/` ссылаются на `../index.html` и на `../cyberpapa-hero.jpg`. Если что-то переместить — сломается.

---

## 🚀 Как выложить в интернет (Netlify Drop, 2 минуты)

1. Откройте https://app.netlify.com/drop
2. Перетащите **всю папку** `kiberpapa-site/` в окно браузера
3. Получите ссылку вида `https://kiberpapa-xyz.netlify.app` — работает сразу
4. Позже можно привязать свой домен (например, `cyberdad.ru` или `kiberpapa.ai`)

**Альтернативы:**
- Vercel (vercel.com) — аналогично, drag-and-drop
- GitHub Pages — бесплатно, нужен аккаунт GitHub
- Reg.ru / Beget / Timeweb — российский хостинг, нужен домен (обязательно при подключении аналитики — см. ниже)

## ⚖️ Юридический чек-лист перед первым платежом

### 🟥 Критично:

1. **Добавить ОКВЭДы у ИП** (сейчас основной 73.11 — реклама)
   - Через Госуслуги → добавить 62.09, 63.11.1, 63.99

2. **Подать уведомление в Роскомнадзор** об обработке ПДн
   - https://pd.rkn.gov.ru/operators-registry/notification/
   - Бесплатно, онлайн через ЕСИА
   - Штраф за отсутствие — до 300 000 ₽ для ИП

3. **Юрист проверил 4 документа** в папке `legal/`

4. **Документы опубликованы на сайте** (выложен архив на хостинг)

5. **В боте** перед первой платной функцией — экран «Я согласен с офертой и политикой конфиденциальности» с кнопкой (не предвыбранной по умолчанию)

6. **Кризисный протокол** встроен в системный промпт бота + работает keyword-классификатор

7. **ЮKassa** подключена, тестовый платёж прошёл

### 🟧 Желательно:

8. **Проверить название** в Роспатенте (классы 9, 42, 45)
9. **Российский хостинг**, если будете подключать Яндекс.Метрику

## 🔧 Технические заметки

- **Статический сайт**: HTML + CSS + немного JS
- **Адаптивен**: мобильная версия ≤760px
- **Зависимости**: только Google Fonts (Fraunces + Nunito)
- **Аналитика и cookie** не подключены

---

## 📞 Если что-то не работает

- **Картинка не видна** — проверьте, что `cyberpapa-hero.jpg` рядом с `index.html`
- **Документы не открываются** — папка `legal/` должна быть рядом с `index.html`, не вложена
- **Шрифты некрасивые** — проверьте интернет, Google Fonts грузится при первом открытии

Удачи с запуском 🚀
